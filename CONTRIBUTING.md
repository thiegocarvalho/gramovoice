# Contributing to GramoVoice

First off, thank you for considering contributing to GramoVoice! It's people like you that make GramoVoice such a great tool.

## Where do I go from here?

If you've noticed a bug or have a request for a feature, please [open an issue](https://github.com/thiegocarvalho/gramovoice/issues).

### Fork & create a branch

If this is something you think you can fix, then [fork GramoVoice](https://github.com/thiegocarvalho/gramovoice/fork) and create a branch with a descriptive name.

## Setting up your local environment

1. Clone your fork: `git clone https://github.com/YOUR-USERNAME/gramovoice.git`
2. Change directory: `cd gramovoice`
3. Create a python virtual environment: `python -m venv venv`
4. Activate the virtual environment:
   - Linux / macOS: `source venv/bin/activate`
   - Windows: `venv\Scripts\activate`
5. Install dependencies: `pip install -r requirements.txt`
6. Run the CLI/GUI mode: `python main.py` or Run the API mode: `python main.py --api`

## Submitting a Pull Request

1. Ensure any changes or new features have been fully tested.
2. Ensure your code follows PEP-8 styling standards.
3. Open a Pull Request from your branch to the `main` branch.
4. Describe your changes clearly in the PR description.

Thank you for contributing!
