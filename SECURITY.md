# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| v1.2.x  | :white_check_mark: |
| < v1.2  | :x:                |

## Reporting a Vulnerability

If you discover a potential security issue in this project we ask that you notify the repository owner directly. Please do **not** create a public github issue.

Instead, please send an email with a detailed description of the issue, the steps you took to create the issue, and the affected versions.

We will evaluate the issue and release a patch as soon as possible.
